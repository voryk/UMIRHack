# UMIRHack
# Дизайн-квиз (Frontend)

Простой интерактивный квиз для сбора информации о проекте (тип помещения, площадь, бюджет и контакты пользователя). Реализован на чистом HTML, CSS и JavaScript без сторонних библиотек.

## 🚀 Функциональность

* Пошаговая форма (6 шагов)
* Индикатор прогресса
* Разные типы ввода:

  * Радио-кнопки
  * Числовые поля
  * Слайдер (range)
  * Выпадающие списки
* Валидация обязательных полей
* Анимации переходов между шагами
* Простая обработка формы (alert)

## 📁 Структура проекта

```
quiz.html   # Основной файл с разметкой, стилями и логикой
```

## ▶️ Запуск

1. Скачайте репозиторий
2. Откройте файл `quiz.html` в браузере

Никакая установка зависимостей не требуется.

## 🛠️ Технологии

* HTML5
* CSS3 (градиенты, анимации, flexbox)
* Vanilla JavaScript

## 📌 Возможные доработки

* Подключение отправки данных на сервер (API)
* Добавление валидации форм (например, через regex)
* Сохранение ответов (localStorage)
* Адаптация под мобильные устройства (доп. оптимизация)
* Интеграция с CRM
